Back-end

Because beauty isn't only at front-end, this M Boilerplate includes beautiful technologies at its back-end. Among the most interesting you'll find:

  • Python 2.7.5, well you know Python don't you?,
  • NDB 1.0.10, the best no-sql database API with unlimited scalability,
  • Jinja2 2.6, a fully featured template engine for Python,
  • WTForms-1.0.2, forms validation keeping user interaction secure and flexible,
  • Babel-0.9.6, renders the site in multiple languages, and
  • WebApp2 2.5.2, lightweight Python web framework, best for the GAE Python runtime.

Front-end

With M Boilerplate everything will be shinny because it has been brought from the future, where the internet is pretty and organized. In this future you'll find:

  • HTML5Boilerplate, yes, the HTML that rocks !,
  • Modernizr 2.6.1, modernize says it all,
  • jQuery 2.1.1, the JS for Safari =),
  • MaterializeCSS 3.1, beauty at its best,
  • Polymer, web components or the future, now,
  • HighCharts, D3, ChartJS, ChartistJS, Data Tables, among other data visualization libraries.

Integrations

As if it wasn't enough, M Boilerplate has been prepared for you to easily integrate popular technologies. Among the most interesting you'll find:

  • Bit.ly and Goo.gl, for shortlink generators,
  • Slack, to receive chat notifications everytime you have a new user so that you can tell them how much you care,
  • Zendesk, to get feedback like a zen,
  • Sendgrid, ran out of 20,000 monthly emails quota?, no worries you'll be ready to use your second bucket,
  • Mailchimp, to get email subscribers to your blog posts,
  • CartoDB, geo-reference, postGIS + postgresql at its best.

  • Innate benefits (Google AppEngine)

    (1) Ideal choice for getting your idea out quickly and seeing if there is a viable market (2) Zero to App in a few minutes, no other provider out there gets you set up and developing in as little time (3) Admin interface that gives access to everything you need (4)Excellent documentation as well as active stack overflow community (5) Free for vast quotas, but you can calculate how much you need to pay when you have proven there's a viable market (6) Amazing performance and real-time on-the-go monitoring (7) Custom domain if you need or free appspot.com (8) Scaling is automagical.

  • Essential features (GAE Boilerplate)

    (1) Authentication: login, logout, register (2) Login via your favorite social network (3) Reset password (4) Update user profile (5) Contact form (6) Client side and server side form validation (7) Automatic detection of user language (8) Support for many languages (9) Visitors log (10) Notifications and confirmation for users when they change their email or password (11) Responsive design for viewing on PCs, tablets, and mobile phones (12) Mobile device recognition (13) Unit testing (14) Error handling (15) Basic user management features available under /admin/ for GAE Administrators.

  • Security and passwords (GAE Boilerplate)

    (1) SSL is enabled site wide by adding secure: always to the section: - url: /.* in app.yaml (2) SSL either requires a free google app engine *.appspot.com domain or a custom domain and certificate (3) Alternatively SSL can be enabled at a controller level via webapp2 schemes. Use the secure_scheme provided in routes.py (4) It is recommended to enable ssl site wide to help prevent session hijacking (5) Passwords are hashed and encrypted with SHA512 and PyCrypto (6) Forms are secured through cross-site request forgery protection.

  • Additional perks (MBoilerplate)

    (1) Dropbox-inspired referrals system for users to invite with their unique link (2) Rewards system for you to praise users' loyalty (3) Pre-loaded web components for easier chart generation (4) Beautiful email templates for a bunch of situations (5) Pretty server-client messaging through toast, dismissible, swipeable notifications (6) API Key - based handlers for complete REST API (7) Enabled Endpoints for Google-based API as mobile backend for running cloud endpoint clients such as Android and iOS apps (8) Image handling ready ! Dispatching files using Google's NDB Blob Properties and Google Images API (9) Konami-code surprise for the fun of it (10) Super easy blogging tool (11) Profile connection to favorite social networks (12) Beautiful examples using Polymer and CartoDB (13) to be continued...
